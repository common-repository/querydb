=== QueryDB Plugin ===
Tags: database, query, widget
Requires at least: 2.0.0
Tested up to: 3.0.1
Stable tag: 1.0

== Description ==

This widget will return a table containing data from an SQL you provide 

== Installation ==

Copy the file querydb.php into the wp-content/plugins folder of your WordPress installation. 
After that go to Administration page, choose Plugins. QuerDB Plugin should be in the list. 
Activate it.

== Frequently Asked Questions == 

None at present

== Using the plugin ==

Drop you widget in, give it a title and an SQL statement. That's it
